# ☕ Cafe POS System (Django)

A modern, fast, and customizable **Point of Sale (POS) system** built with Django — designed for cafes, restaurants, and small food businesses to manage orders, kitchen workflow, and daily operations efficiently.

---

## 🚀 Live Demo

🌐 https://modeltownclub.pythonanywhere.com/

---

## 🎯 Who is this for?

* ☕ Cafes & Coffee Shops
* 🍔 Fast Food Restaurants
* 🍽 Small to Medium Restaurants
* 🧾 Businesses needing a custom POS solution

---

## ✨ Key Features

* 🧾 **Order Management**
  Create, update, and track customer orders in real-time

* 🍳 **Kitchen Dashboard**
  Separate interface for kitchen staff to manage orders efficiently

* 📋 **Menu Management**
  Easily add/edit items, categories, and pricing

* 🪑 **Table Management**
  Handle dine-in operations with table tracking

* 📋 **Membership Management**
  Members independent login, Online Order, and Order status checking

* ⚡ **Fast & Responsive UI**
  Designed for smooth real-time usage in busy environments

## ⚙️ Installation (Local Setup)

```bash
git clone https://github.com/massab1/cafe-pos-system.git
cd cafe-pos-system
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
